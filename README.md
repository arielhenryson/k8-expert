# k8-expert

This is a Chainlit application.

## Setup

1. **Create and activate the conda environment:**
   ```bash
   conda create -n k8-expert python=3.12 -y
   conda activate k8-expert
   ```

2. **Install dependencies using uv:**
   ```bash
   pip install .
   ```

3. **Run the application:**
   ```bash
   chainlit run app.py
   ```
